# Project NakamFood

Laravel WebApp untuk memesan makanan daring

### Requirements

- Laravel 6
- Composer
- Artisan

### Instalasi

Fork repository ini dan clone hasil forknya.

Setelah itu, duplikasi file .env.example menjadi .env dan atur database
```
  APP_NAME=NakamFood
  ...
  DB_USERNAME=root
  DB_PASSWORD=
```

Lalu jalankan perintah,
```sh
$ composer update
```
